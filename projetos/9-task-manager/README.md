# 9-task-manager

Gerenciador de tarefas em Java com suporte a criação, conclusão e remoção.

Inclui:

- Classe `Task` com status/priority/timestamps;
- `TaskManager` para armazenar e manipular objetos;
- Guias de ajuda nos comentários.

Compile com `javac` e execute `TaskManager` da linha de comando.

### Design
- `Task.java` define campos como `status`, `priority` e timestamps;
- `TaskManager.java` inclui métodos para adicionar, listar, filtrar e marcar tarefas concluídas.

### Uso prático
O projeto demonstra conceitos de coleções Java (`ArrayList`), manipulação de objetos e pode ser estendido com persistência em arquivo ou banco de dados.
